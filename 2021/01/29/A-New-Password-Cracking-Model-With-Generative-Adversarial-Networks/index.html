A New Password Cracking Model With Generative Adversarial Networks

发表于 2021-01-29 分类于 论文 阅读次数：
本文字数： 4.6k 阅读时长 ≈ 4 分钟

《A New Password Cracking Model With Generative Adversarial Networks》论文阅读记录

摘要

  • 使用 RNN+IWGAN —— 比PassGAN效果好
  • 效果提高 5%-10%

介绍

  • 加密软件：VeraCrypt【15】
  • 三种破解密码方法：
    • 暴力破解——组合一定数量字符，直到 hash 相同
    • 基于字典的破解【19】——尝试使用哈希算法在恒定时间内破解密码，而不考虑密码的字长，成功率取决于攻击字典；攻击范围受限于攻击字典持有的密码候选数量
    • 混合攻击【19】——支持攻击字典中候选密码的转换来提高基于字典的攻击的破解性能
  • JTR【1】与 Hashcat【2】支持以上三种攻击方法：将内置规则（如 best64）应用于泄漏密码字典（如 Rockyou【3、7】）
    • 通过分析数千万人使用的泄漏密码来创建反映所有密码变体的规则文件是困难的
    • 密码破解规则反映的是一个专家的个人经历，所以规则的破解性能并不一致
  • 两种省时的分析密码结构方法：
    • 基于概率分析
    • 基于深度学习
  • 提出效果优于 PassGAN 的算法，并测试不同的训练参数

相关研究

马尔可夫和上下文无关语法

  • 人们在整个密码搜索空间中使用最多的密码空间仅限于在创建密码时容易记住的密码，并且此区域中的密码遵循一定的字母数字组合概率分布。创建了一个密码生成规则，该规则将各种字母和数字的组合表示为正则表达式，并定义每个组合的概率
  • PCFG 思想在最近有提高与修改【14、22】
  • 实验中，使用 PCFG 的密码猜测方法比使用 Hashcat 内置规则的基于字典的攻击具有更高的破解成功率

深度学习方法

  • RNN【6】
    • 泄露数据作为训练集，以字符为单位生成新密码
    • 构成密码的字符基于先前选择的字符
  • PassGAN【12】
    • GAN【10】的原理
    • WGAN【4】提出更稳定的训练方法

提出的模型

  • 改变鉴别器和生成器中的深度学习模型（r-PassGAN）
    • CNN不是最好的文本生成方法——将CNN改为RNN
  • 改变PassGAN模型的结构（r-PassD2CGAN）
    • 两个判别器，均为 IWGAN 损失函数——【20】
    • D1 在真实和虚假的密码中检测真实密码，D2 在真实和虚假密码中检测虚假密码
    • G 同时欺骗两个判别器
    • 定义了每个鉴别器的损失之和，训练D1和D2使损失之和最小化
    • G经过训练，使损失总和相对于鉴别器的成本最大化

实验

  • 度量简单的破解成功率和破解密码的可扩展性来评估密码的性能

  • 训练参数

    • 训练 20k 轮
    • G/D 训练比例从 10:10 到100:10
    • 为评估测试结果，在 RockYou 应用 N-gram（N=3，4）Jensen-Shannon 散度（JSD）值 vs 每个基于RNN的模型生成的字典

  • 训练数据

    • 从 RockYou 中排除了 unicode 字符
    • 分析 RockYou 字典中密码的长度。然后使用 RockYou 的所有密码（大约1400万，密码长度：1-32）
    • 对于基于密码长度的每个模型的性能评估，根据长度将密码分为三个部分（1-8、9-15和16-32）
    • 使用 RockYou 为每个密码长度部分创建训练字典
    • 将密码按长度分为三个部分进行实验的目的是确定哪个模型在每个密码长度部分具有最佳性能

  • 密码破解

    • 测试集为LinkedIn
    • Hashcat 应用 best64 规则
      • 为了在类似于实际领域的条件下测量猜测密码的破解性能，此时往往使用规则 best64
      • 使用深度学习模型生成数十亿个密码需要大量时间。因此，即使生成少量密码，也可以通过应用 best64 规则成功破解许多与简单转换相关的密码

评估

  • 字典质量

    • 训练字典和生成字典之间的相似性——n-gram JSD

    • 生成字典中的 unique guessing password ——密码的冗余度

  • 破解表现

    • 讨论了 r-PassGAN 和 r-PassD2CGAN 对 PassGAN 的累积密码破解性能和破解区域扩展能力

  • 总破解密码

    • 与训练轮次有关

    • 与密码长度有关

  • 破解的扩展性

    • 分析了在所有提出的模型中，只有该模型可以破解的密码数与该模型没有破解的密码数之比

    • 每个模型都有 unique cracked password

    • RNN模型之间的差别非常小

总结

  • 使用 RNN 作为基本 DNN

  • 优化训练的 G/D 训练比是通过实验得到的

  • r-PassD2CGAN 在训练轮次较少（如10k）内表现出可接受的破解性能

  • 每个模型都有独有的破解密码
