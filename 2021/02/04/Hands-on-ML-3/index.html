《Hands-on Machine Learning》第二部分阅读笔记（3）用 tf 自定义模型训练网络

TF 加载和预处理数据

Data API

  • from_tensor_slices(X)创建一个元素都是 X 的切片（沿第一维）的 tf.data.Dataset

链接转换

  • 一旦有了数据集，就可以通过调用转换方法对它做各种转换

  • dataset = dataset.repeat(3).batch(7)

    • 为了让所有 batch 都具有完全相同的大小，需要丢弃最后一个 batch，设置参数drop_remainder=True

  • 通过map()将任何转换应用到 item

  • 通过apply()将任何转换应用到数据集

  • 通过filter()过滤

  • 仅查看数据集中的几个项目：dataset.take(3)

打乱数据

  • 当训练集中的实例独立同分布，梯度下降法效果最好——需要 shuffle data

    • 设置随机数种子

  • 将源数据分成多个文件，然后在训练过程中以随机顺序读取

  • 可以随机选择多个文件，并同时读取，交错它们的行

    • train_filepaths是一个列表，里面为源文件的路径
    • list_files()返回一个打乱文件路径的数据集
    • interleave()一次读取 5 个文件——提取 5 个文件路径，对于每个路径，调用传入的函数；在这 5 个数据集之间循环，每次读取一行，直到所有数据集都用完为止
    • skip()跳过第一行
    • 将num_parallel_calls参数设置为想要的线程数，实现并行

预处理数据

  • 举例：

    • 假设预先计算了训练集中每个特征的均值和标准差
    • 取一个CSV行，使用tf.io.decode_csv()解析，其中第二个参数为文件中每列默认值数组
    • decode_csv()返回一个标量张量列表，需要调用tf.stack()把标量张量堆叠成一个一维张量
    • 缩放输入特征

合并以上过程

  • 创建并返回一个数据集，该数据集从多个 CSV 文件中加载数据，shuffle、预处理并 batch

预取数据

  • prefetch：当训练算法处理一批数据时，数据集已经在并行处理下一批数据
